以下はキーボードの動作のために必須ですが、キットには含まれていません。  
別途ご購入ください。

* キットに含まれない必須パーツ

    | 品目 | 数量 |
    |  :-  |  -:  |

    | キーキャップ  | - |
    | 2U スタビライザー  | 3 または 2※ |
